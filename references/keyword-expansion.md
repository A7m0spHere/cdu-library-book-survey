## Search Practice

- Search broad Chinese terms first for local holdings.
- Search English terms for imported/translated books.
- Search rare methods directly: RDKit, QSAR, molecular docking, virtual screening.
- If a keyword returns 0 results, search its Chinese translation and broader parent topic.
