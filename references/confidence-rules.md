# Confidence Rules

Every recommended book needs confidence.

| Level | Meaning |
|---|---|
| A | OPAC detail page, holdings status, and location/shelf data are verified |
| B | OPAC result and detail page are verified, but precise shelf data is missing |
| C | External data assists judgment, or holdings/version information is incomplete |
| D | Based on user goal and partial overview; explicitly mark as judgment |

## Evidence Labels

Allowed evidence labels:

- OPAC检索结果
- OPAC详情页
- 馆藏条码页
- 图书定位页
- 目录/摘要
- 外部出版信息
- 人工判断
- 缓存数据

If live OPAC cannot be verified, do not use confidence A.
