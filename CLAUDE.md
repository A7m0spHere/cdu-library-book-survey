# Claude Code Entry

Use this repository as a neutral Skill package, not as a one-off prompt.

1. Read `README.md` and `references/workflow.md`.
2. Use `scripts/opac_survey.py` if the OPAC is reachable.
3. Load the relevant reference files for ranking, confidence, hallucination prevention, and the final report template.
4. Do not fabricate library facts. Mark `[未核验]`, `[缓存数据]`, `[外部资料]`, or `[人工判断]` where applicable.
5. Produce the final report using `references/report-template.md`.
